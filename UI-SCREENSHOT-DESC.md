### Color Scheme
- **Background:** Dark theme (#1a1a2e navy/charcoal)

- **Secondary:** Purple (#7f5af0)
- **Success:** Green (#2cb67d)
- **Warning:** Orange (#ff9e64)
- **Text:** Light gray (#e5e5e5)

### Typography
- **Font:** Inter (sans-serif)
- **Headings:** 600-700 weight
- **Body:** 400-500 weight
- **Monospace:** For event IDs, npubs

### Components
- **Search Bar:** Large, prominent, auto-focus
- **Result Cards:** Clean cards with hover effects
- **Badges:** Color-coded by event kind
- **Quality Stars:** Visual quality indicator
- **Loading States:** Skeleton screens
- **Empty States:** Helpful illustrations

### Interactions
- **Search:** Real-time suggestions as you type
- **Filters:** One-click toggle
- **Results:** Click to view full document
- **Pagination:** Smooth transitions
- **Copy:** One-click copy for event IDs, npubs
