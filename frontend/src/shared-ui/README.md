## Usage
Import the CSS once in each product frontend (e.g., in `src/index.css`).

```css
@import "../../../shared-ui/shared-ui.css";
```

Then use classes like `.ui-app`, `.ui-card`, `.ui-cta`, `.ui-table`, `.ui-status`.
